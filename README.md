# 步骤 #


## 一、跟踪项目代码 ##

1. 用自己账号登录 Github

2. 打开 https://github.com/lvye1221/bootstrap-course.git

3. 点击 Fork 按钮，如下图所示：

    ![](snap/1.png)

4. 在自己账户上，克隆 自己这个项目 到本地（以后在自己的项目中进行练习、提交）

5. 【以后】【如果需要】提交代码到原始处，通过 创建一个 “Pull Request”来实现

    ![](snap/2.png)
